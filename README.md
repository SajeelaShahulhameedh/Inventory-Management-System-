# Inventory Management System

## Project Overview
A dynamic web application for managing inventory, products, and suppliers using PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- Product Management (Add, View, Edit, Delete)
- Inventory Tracking (Stock Management)
- Supplier Management
- Dashboard with Analytics
- Responsive Design (Desktop, Tablet, Mobile)
- Form Validation
- OOP-based Architecture

## Technologies Used
- **Backend**: PHP (OOP)
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript
- **Server**: Apache/XAMPP

## Installation
1. Clone the repository
2. Create a MySQL database using `sql/database.sql`
3. Update `config/database.php` with your credentials
4. Place project in your web server root
5. Access via `http://localhost/Inventory-Management-System`

## Folder Structure
```
Inventory-Management-System/
├── config/          (Database configuration)
├── classes/         (OOP classes)
├── pages/           (All web pages)
├── assets/          (CSS, JS, Images)
├── sql/             (Database schema)
└── index.php        (Main entry point)
```
